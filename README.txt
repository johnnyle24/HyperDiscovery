# DomainSpecificHypernymDiscovery

How to run program:

python3.6 Driver.py <corpusFilename> <conceptFile> <patternsFile>

Example
python3.6 Driver.py ../Data/2A_med_pubmed_2/2A_med_pubmed_2_{0}.txt ../SemEval2018-Task9/training/data/2A.medical.training.data.txt ../MinedData/suggestedPatterns.txt

Or to run with default params:

python3.6 Driver.py
